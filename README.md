# Sean Millard — Portfolio

Personal portfolio site showcasing my work as a full-stack web developer.

## What it is

A fast, minimal portfolio built to present selected projects, background, and credentials to potential clients. No CMS, no templates — designed and built from scratch.

## Problem it solves

Most portfolio templates look like portfolio templates. This is a custom build that reflects how I actually work: clean structure, good defaults, no unnecessary complexity.

## Tech stack

- [Astro](https://astro.build) — static site framework
- Tailwind CSS — utility-first styling
- Vanilla JS — scroll reveal, theme toggle
- Deployed on Netlify
